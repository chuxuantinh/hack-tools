Versao do python 2.7
Passo-a-passo exucução do programa:
1- Execute o zombie.py nas maquinas bot excutando o comando "python zombie.py ipCliente" no terminal
2- Execute o client.py na maquina pai excutando o comando "python client.py" no terminal
3- Envie pelo cliente o comando "synflood ipFonte ipDestino" para ativar o ataque synflood
ou envie pelo cliente o comando "slowloris ipDestino numeroDeSocketsPorBot" para ativar o slowloris.

Observação: o ataque slowloris so funciona em servidores Apache executando em Windows
